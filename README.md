# EvalPro — Private Investment Project Monitoring Platform 🚀

## Overview
Full-stack web/mobile platform for monitoring and evaluating private 
investment projects. Developed as final year engineering project at 
Medianet (March 2025 – September 2025).

## Project Description
Development of a web and mobile platform for monitoring and evaluating 
private investment projects.

- Analyzed requirements and designed a modular, scalable architecture 
  (frontend, backend, DevOps).
- Built the web frontend with Angular and the backend with Spring Boot 
  (data management + security).
- Automated deployments with a CI/CD pipeline using Jenkins and Docker.
- Delivered continuous delivery on Kubernetes using ArgoCD, with testing 
  and CI for quality.
- Integrated AI features to enhance the platform, such as intelligent 
  assistance and automated content generation.

## Tech Stack

Frontend   : Angular
Backend    : Spring Boot
Database   : MySQL
CI/CD      : Jenkins, Docker
GitOps     : ArgoCD
Orchestration : Kubernetes
AI         : OpenAI API
Cloud      : Azure

## Architecture
Angular → Spring Boot API → MySQL
GitHub → Jenkins → Docker → Kubernetes → ArgoCD
